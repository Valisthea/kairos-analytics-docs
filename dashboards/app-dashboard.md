# App Dashboard

Real-time behavioral analytics for your application.

---

## URL

https://kairosanalytics.org/dashboard

## Features

21 widgets organized by plan tier:

- **Real-time feed** -- Live event stream with anchoring status
- **KPI cards** -- Sessions, events, unique wallets, conversions
- **Timeline** -- Event distribution over time
- **Category breakdown** -- Events by type (page, UI, wallet, transaction)
- **Session explorer** -- Individual session details
- **Top pages** -- Most visited pages
- **Heatmap** -- Activity by day and hour
- **Geographic distribution** -- Events by country
- **Journey analysis** -- User flow paths
- **Retention** -- User return rates
- **Conversion funnel** -- Step-by-step drop-off analysis

## Audit and Security

12-tab security dashboard at /audit-security:

Conversion, Integrity, Reliability, Growth, UX, Phantom Revenue, Temporal Heartbeat, Wallet Genome, K-PPE Security, Sentinel.

Each audit is cross-linked with enriched findings, recommendations, and $ ROI impact.
