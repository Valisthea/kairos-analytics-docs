# Audit and Security

12-tab security and performance audit at /audit-security.

---

## Tabs

| Tab | Description |
|-----|-------------|
| **Overview** | All modules with narrative descriptions and cross-link indicators |
| **Action Board** | Prioritized findings sorted by dollar ROI |
| **Conversion** | Funnel analysis with behavioral patterns |
| **Integrity** | Sybil detection, corrected DAU vs reported |
| **Reliability** | Transaction error patterns and spike analysis |
| **Growth** | Quick Ratio decomposition (new/reactivated/churned/net) |
| **UX** | Rage clicks, frustration events, friction signals |
| **Phantom Revenue** | Dollar-priced revenue leakage with 6-cause classification |
| **Temporal Heartbeat** | 168-hour behavioral quality heatmap |
| **Wallet Genome** | Behavioral species classification with DNA strips |
| **K-PPE Security** | 7-layer cryptographic security status |
| **Sentinel** | Live security scanner with 20+ checks |

## Cross-audit Intelligence

Each finding links to related audits. For example, a slippage bug in Reliability links to Phantom Revenue (lost dollars), Genome (whale extinction risk), and Heartbeat (time impact).
